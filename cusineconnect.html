Project Overview

Cuisine Connect is a restaurant booking website designed to streamline the reservation process for diners and restaurants alike. Built using HTML, CSS, JavaScript, React, and JSON server, the site offers a responsive and user-friendly platform for browsing restaurants, viewing menus, and making reservations with ease
